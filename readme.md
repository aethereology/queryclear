# queryclear

**We make your website easier for search engines and AI answer engines to
understand, trust, and recommend.**

queryclear is a **modern SEO service for the AI search era** (sometimes called
GEO / AI Search Optimization) — we optimize websites so search engines and AI
answer engines like ChatGPT, Claude, Perplexity, Gemini, Google AI Overviews,
and Bing Copilot can better crawl, understand, summarize, trust, and cite a
business.

queryclear is a **SparkCreatives Inc.** brand.

> **Status:** LIVE at [www.queryclear.com](https://www.queryclear.com).
> This repo contains the site (Next.js) plus strategy and product docs
> (see `roadmap.md`).

## What we do

- **Free AI Search Snapshot** — a quick plain-English review of your site's
  search clarity, technical foundation, and biggest opportunities.
- **AI Search Audit ($497)** — a deeper scored report: prompt testing, page
  review, technical findings, local visibility review, and a prioritized fix
  roadmap.
- **Website Upgrade (from $2,500)** — we implement the fixes (schema, sitemap,
  metadata, FAQs, structure, crawlability) on your existing site.
- **Modern Search Website Build (from $6,500)** — a new, AI-search-ready website.

## What we promise

Better technical readiness, clearer content, valid structured data,
crawlability, and honest measurement. We do **not** promise guaranteed rankings
or guaranteed AI citations, and we never invent business details, reviews, or
testimonials.
